# HOOPS AI MCP — デモ ナレーション（日本語）

---

## スライド 1 — HOOPS AI

HOOPS AI は、Tech Soft 3D が提供する 3D CAD 専用の機械学習フレームワークです。30 以上の 3D CAD フォーマットの読み込みから、データセット構築、モデル学習・推論まで、3D CAD と AI に必要なすべてをひとつのツールキットで提供します。

主な機能は 4 つです。まず、HOOPS Exchange で 30 以上のフォーマットをネイティブに Python 環境で読み込めること。次に、3D CAD データからエンコードしたデータセットをスクリプトで自動構築できること。加工フィーチャー認識や類似形状検索といった学習済みモデルの提供と、Python API によるカスタムモデルの学習・推論。そして HOOPS Visualize for Web との統合による 3D ビューアー上へのリアルタイム可視化です。

---

## スライド 2 — Title

この動画では HOOPS AI を実用的に活用するための「HOOPS AI MCP」をご紹介します。これは、3D CAD データをインテリジェントに解析するプラットフォームです。3D CAD ビューアー、B-Rep 解析、加工フィーチャー認識、そして類似形状検索という 4 つの主要機能を、FastAPI による REST API をClaude Desktop から自然言語で操作できる MCP サーバーでラップしたインターフェースで提供します。

---

## スライド 3 — Architecture

システムは 4 層構造になっています。土台となるのは Tech Soft 3D 製の HOOPS AI で、3D CAD ファイルの読み込み・形状エンコード・機械学習による推論を担います。その上に FastAPI の WebAPI が乗り、HOOPS AI の機能を REST API として公開します。FastAPI は Python で書かれた高速な Web フレームワークで、HOOPS AI の Python API をそのまま活用しながら Web API を構築できるのが大きな強みです。MCP サーバーはその WebAPI をラップし、Claude Desktop が自然言語でツールを呼び出せるようにブリッジする役割を持ちます。そして最上位の Claude Desktop が、チャットAIとして MCP ツールを自律的に呼び出し、3D CAD 解析タスクをエンドツーエンドで実行します。

---

## スライド 4 — Features

4 つの機能を順にご説明します。まず「3D CAD Viewer」は、STEP・SolidWorks・CATIA・NX など 30 以上のフォーマットをブラウザで 3D 表示します。次に「B-Rep Analysis」では、面隣接グラフの生成とフェイス・エッジ属性の抽出が可能です。「MFR」は学習済み ML モデルを使い、穴・スロット・ポケットなど 24 種の加工フィーチャーを自動認識し、結果をビューアー上に色付けで可視化します。最後の「3D CAD Similarity Search」は、HOOPS Embeddings で形状を特徴ベクトルに変換し、FAISS インデックスで類似形状を高速検索します。

---

## スライド 5 — WebAPI Endpoints

WebAPI は機能ごとにエンドポイントを整理しています。3D CAD Viewer の起動・終了、B-Rep の面隣接グラフ生成と属性抽出、MFR のファイル検索・サムネイル取得・ラベル一覧・推論実行・ビューアー色付け、そして類似形状検索の合計 11 エンドポイントを提供します。

---

## スライド 6 — MCP Tools

MCP Tools は Claude Desktop から呼び出せるツール群です。WebAPI の各機能に対応した 11 個のツールを提供しており、Claude は自然言語の指示だけでビューアーを起動したり、加工フィーチャーを認識したり、類似形状を検索したりできます。エンジニアがコードを書かなくても、対話的に 3D CAD 解析を進められるのが最大の特長です。

---

## スライド 7 — Summary

まとめです。HOOPS AI MCP は、REST API と チャットAI連携というアプローチで 3D CAD インテリジェンスを提供します。B-Rep 解析と加工フィーチャー認識により 3D CAD データを深く理解し、HOOPS Embeddings と FAISS を組み合わせた類似検索ではスコア 0.99 以上の高精度を実証しました。3D CAD データ活用の新しい可能性を、ぜひご体験ください。
